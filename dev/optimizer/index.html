Optimizer

The optimizer updates the parameters of a model from the gradients computed by the Autograd engine. MicroGPT provides Adam, which maintains per-parameter moment estimates across update steps. See the original paper, Adam: A Method for Stochastic Optimization, for the underlying algorithm.

A training step has three parts: zero the gradients with zero_grad!, run a forward/backward pass to populate them (see backward!), and apply one update with step!.

Each call to step! advances the optimizer's timestep, which is used for the bias correction of the moment estimates, so the moments accumulated across iterations carry over between steps.

Hyperparameters

Adam is constructed from a flat Vector of AValue parameters and accepts the standard Adam hyperparameters as keyword arguments:

KeywordDefaultMeaning
α0.01learning rate
β10.85first-moment decay
β20.99second-moment decay
ϵ1e-8numerical stabilizer
opt = Adam(params; α=0.001, β1=0.9, β2=0.999)
